# 真央ゼミWebサイト
真央ゼミのWebサイト用リポジトリです。
GitHub Pages機能を使って公開されたページは[こちら](https://elmas3.github.io/mao-seminar/)から閲覧できます。
https://elmas3.github.io/mao-seminar/

# ディレクトリ構成
```
seminar/
├─ index.html        # トップページ
├─ member.html       # メンバー紹介ページ
├─ work.html         # ゼミ活動紹介ページ
├─ README.md
├─ images
└─ css/
    └─ common.css
```

# 備考
本リポジトリは「わかばちゃんと学ぶGit使い方入門」の練習用として使用するリポジトリです。
